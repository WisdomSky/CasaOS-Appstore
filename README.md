# WisdomSky's Casaos Appstore

WisdomSky's custom collection of CasaOS Apps

## Requirements

* CasaOS [0.4.4+](https://blog.casaos.io/blog/23.html)

## Install AppStore

```bash
casaos-cli app-management register app-store https://paodayag.dev/appstore.zip
```
